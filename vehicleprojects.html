Vegetable Oil Fuel

About The Project:

This project goes in tandem with the 1993 Volkswagen Golf. The Golf has an older diesel-powered motor that is capable of running alternative fuels. With the proper filtering procedures and patience, it is possible to run waste vegetable oil rather than diesel. The hardware is prepared, currently, the limiting factor is sourcing the waste oil from restaurants. This is a work in progress!
